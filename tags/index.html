Linux操作系统一

最近在学操作系统需要用到命令行 码一下记录一点
操作 命令
创建 touch filename
编辑 vi filename
查看 cat filename
复制 cp filename copyfile
重命名 mv filename newfile
删除 rm filename filename2
统计词数 wc filename
数字 说明 权限
0 没有任何权限 —-
1 执行权限 —x
2 写入权限 -w-
3 执行权限和写入权限：1 (执行) + 2 (写入) = 3 -wx
4 读取权限 r—
5 读取和执行权限：4 (读取) + 1 (执行) = 5 r-x
6 读取和写入权限：4 (读取) + 2 (写入) = 6 rw-
7 所有权限: 4 (读取) + 2 (写入) + 1 (执行) = 7 rwx
可以使用ls -al来查看当前目录下的所有文件列表。

查看权限

$ls -l testfile
-rwxrwxr–1 amrood users 1024Nov200:10 testfile

增加权限

$chmod o+wx testfile
$ls -l testfile
-rwxrwxrwx 1 amrood users 1024Nov200:10 testfile

删除权限

$chmod u-x testfile
$ls -l testfile
-rw-rwxrwx 1 amrood users 1024Nov200:10 testfile

指定权限

$chmod g=rx testfile
$ls -l testfile
-rw-r-xrwx 1 amrood users 1024Nov200:10 testfile

同时使用多个符号

$chmod o+wx,u-x,g=rx testfile
$ls -l testfile
-rw-r-xrwx 1 amrood users 1024Nov200:10 testfile
如果你熟悉macos终端的命令方法那么这个命令使用上手起来挺简单的。
